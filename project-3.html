Project Overview

Fast Food Ordering App is a web application developed as part of a university coursework project. The goal of the project is to create a reliable and efficient system for online food ordering, specifically tailored for fast food restaurants. The application allows users to browse the menu, customize orders, easily place them online, and track their status.

The system features a structured backend built with Java Spring Boot, which handles order management and stores customer data using the PostgreSQL database. For displaying information and interacting with users, the Thymeleaf template engine is used. All development processes were carried out in the IntelliJ IDEA environment with version control managed through GitLab. Testing was implemented using JUnit.

The project enabled me to master modern approaches to web application development with a focus on user convenience and process automation.
