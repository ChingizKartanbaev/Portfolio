Project Overview

Pizza App is a web application for online pizza ordering, developed as part of a university coursework project with the goal of creating a convenient and efficient ordering system for fast food restaurants.

The backend was developed using Java Spring Boot, which ensures the reliability and scalability of the application. PostgreSQL was chosen as the database management system to store data about the menu, customers, and order history. Thymeleaf was used for the frontend to dynamically generate content and provide a user-friendly interface.

For payment processing, I created a separate application using REST API, which allows secure transactions and communicates with the main application to confirm payments.
