Project Overview

Cool Band is a web application that serves as a portfolio website for a music band, developed during my internship at GeeksPro. The project involved both backend and frontend development. The goal of the project was to create a modern and user-friendly website that facilitates effective interaction between the band, its fans, and event organizers.

The website provides up-to-date information about the band, upcoming performances, and the option to book the band for events. The primary focus was on usability and an attractive design that matches the bandâ€™s style.

The website features the following sections: a description of the band and its musical style, a gallery with photos and videos, a calendar of past and upcoming concerts, a booking form, and a section for reviews and feedback from organizers and fans.

The backend was implemented using Spring Boot, ensuring a reliable and scalable architecture. Data related to concerts and booking requests is stored in a PostgreSQL database.

The frontend was developed separately using modern web technologies and connected to the backend via a REST API. This approach enabled dynamic content updates and a seamless user experience.

This project allowed me to apply modern development technologies and gain practical experience in creating a full-featured web application. The website helps the band effectively interact with its audience, providing all the necessary information and an easy way to book performances.
